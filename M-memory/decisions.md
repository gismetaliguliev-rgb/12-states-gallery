## Записи

### Решение: Белые стены галереи
**Контекст:** Выбор цвета стен для галереи
**Варианты:**
1. Белые — классика, фокус на фото
2. Серые — современно, но может быть мрачно
3. Тёмные — драматично, но может поглощать внимание

**Выбор:** Белые стены
- Решение Kismet
- Классический галерейный вид
- Максимальный фокус на фотографиях
- Хороший контраст с чёрными рамками
**Статус:** Активно

---

### Решение: Чёрные рамки
**Контекст:** Выбор цвета/стиля рамок
**Варианты:**
1. Чёрные — классика, контраст
2. Белые — минимализм, сливаются со стеной
3. Деревянные — тёплый вид
4. Без рамок — современно

**Выбор:** Чёрные рамки
- Решение Kismet
- Создаёт чёткую границу для фото
- Подходит для драматичного стиля
- Контраст с белыми стенами
**Статус:** Активно

---

### Решение: Three.js (не React Three Fiber)
**Контекст:** Выбор подхода к 3D рендерингу
**Варианты:**
1. Three.js vanilla — прямой контроль, меньше бандл
2. React Three Fiber — React интеграция
3. Babylon.js — много фич, большой размер

**Выбор:** Three.js vanilla
- Сайт Kismet фокусированный, не нужен оверхед React
- Меньше размер бандла = быстрее загрузка
- Больше контроля над оптимизацией
**Статус:** Активно

---

### Решение: JSON-based CMS (не база данных)
**Контекст:** Как хранить конфигурацию галереи
**Варианты:**
1. JSON файл — просто, не нужен сервер
2. SQLite — локальная БД, сложнее
3. Firebase/Supabase — облако, нужен аккаунт

**Выбор:** JSON файл
- Можно хостить на любом статическом хостинге
- Легко редактировать вручную если нужно
- Можно перейти на БД позже если понадобится
**Статус:** Активно

---

### Решение: Одна комната для MVP
**Контекст:** Сложность layout галереи
**Варианты:**
1. Одна комната — просто, фокус
2. Несколько комнат — больше разнообразия, сложнее
3. Бесконечная галерея — другой подход

**Выбор:** Одна комната для MVP
- Сначала заставить работать основное
- Несколько комнат можно добавить позже
- Фокус на core experience
**Статус:** Активно

---

### Решение: Spotlight на каждое фото
**Контекст:** Как освещать фотографии
**Варианты:**
1. Только ambient — просто, но плоско
2. Spotlight на каждое фото — галерейный вид, реалистично
3. Глобальный directional — консистентно, но менее драматично

**Выбор:** Spotlight на каждое фото
- Создаёт аутентичную атмосферу галереи
- Каждое фото получает фокусированное внимание
- Стоит затрат на производительность
- Поддерживает драматичный стиль Kismet
**Статус:** Активно

---

### Решение: Драматичный стиль фотографий
**Контекст:** Общий стиль и настроение
**Варианты:**
1. Нейтральный — универсальный
2. Драматичный — выразительный
3. Минималистичный — современный

**Выбор:** Драматизм + Реализм + Документальность
- Это стиль работ Kismet
- Галерея должна поддерживать, а не противоречить
- Освещение и пространство усиливают драму
**Статус:** Активно

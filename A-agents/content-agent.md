## Гайдлайны по Фото

### Оптимальные Размеры
| Размер в галерее | Разрешение файла |
|------------------|------------------|
| Маленькое (< 1м) | 1200 x 800 px |
| Среднее (1-2м) | 2000 x 1333 px |
| Большое (> 2м) | 3000 x 2000 px |

### Формат
- **JPEG** — для фотографий (80-85% качество)
- **WebP** — альтернатива (меньше размер)

---

## Написание Описаний

### Название (Title)
- Короткое: 3-5 слов
- Эмоциональное, не описательное
- Примеры:
  - "Утренний свет на воде"
  - "Последний день лета"
  - "Тишина города"

### Описание (Description)
- 1-2 предложения максимум
- Включить: место, время, контекст
- Пример: "Снято на рассвете в швейцарских Альпах. Туман рассеялся как раз когда выглянуло солнце."

## Расположение в Галерее

### Схема Стен
```
         СЕВЕР (задняя стена)
            ┌─────────┐
            │         │
   ЗАПАД    │    •    │    ВОСТОК
   (слева)  │  старт  │   (справа)
            │         │
            └─────────┘
         ЮГ (вход)
```

### Позиция X (горизонталь)
- Отрицательные → левая часть стены
- Ноль → центр стены
- Положительные → правая часть стены

### Позиция Y (высота)
- 1.5 - 1.8м → уровень глаз (рекомендуется)
- Выше → для драматического эффекта
- Ниже → для интимного просмотра

---

## Принципы Расположения

1. **Поток** — направляй взгляд естественно
2. **Контраст** — чередуй размеры и темы
3. **Пространство** — не перегружай стены
4. **Фокус** — лучшие работы на видном месте

### Минимальные Отступы
- Между фото: минимум 0.5м
- Большие работы: больше пространства вокруг

# Project Brief — KISMET GALLERY

## Кто Это

**Имя:** Kismet
**Проект:** Фотопроект — пейзажная фотография
**Стиль:** Драматизм, Реализм, Документальная фотография

---

## Суть Проекта

**Одним предложением:** Виртуальная 3D галерея для демонстрации пейзажной фотографии в иммерсивном пространстве.

**Зачем:**
- Выделиться среди обычных grid-портфолио
- Создать запоминающееся впечатление
- Показать работы в контексте (как в реальной галерее)
- Простая ссылка → мгновенный опыт просмотра

---

## Целевая Аудитория

**Основная:**
- Галереи и кураторы
- Ценители искусства
- Коллекционеры

**Вторичная:**
- Обычные люди, интересующиеся фотографией
- Любители искусства
- Посетители из соцсетей

---

## Стиль и Эстетика

### Визуальное Направление
**Референс:** [Kunstmatrix — Strand Sand Meer](https://artspaces.kunstmatrix.com/en/exhibition/15145766/strand-sand-meer)

### Галерея
- **Стены:** Белые (чистый галерейный белый)
- **Рамки:** Чёрные, современный стиль
- **Освещение:** Мягкое, галерейное, с акцентами на фото
- **Пол:** Тёмный (контраст со стенами)

### Настроение
- Драматичное
- Реалистичное
- Документальное
- Минималистичное пространство — фокус на фотографиях

### НЕ делать
- Перегруженный UI
- Отвлекающие элементы
- Сложную навигацию
- Абстрактное/артистичное пространство

---

## Ключевые Функции

### MVP (Первая версия)
1. **3D Галерея**
   - Белый зал для прогулки
   - Фото в чёрных рамках на стенах
   - Управление: WASD + мышь
   - Клик по фото → детали/полный экран

2. **CMS Система**
   - Загрузка фото
   - Позиционирование на стенах
   - Названия и описания
   - Без кода

3. **Производительность**
   - Быстрая загрузка
   - Работает на мобильных
   - Все современные браузеры

### Будущее (Nice to Have)
- Несколько залов/тем
- Фоновая музыка
- VR поддержка
- Форма контакта
- Заказ принтов

## Пользовательский Опыт

### Первый Визит
1. Открыл страницу → "wow" эффект
2. Интуитивное управление → сразу начал исследовать
3. Фото в центре внимания → чистый UI
4. Легко связаться → понятный call-to-action

### Повторный Визит
1. Заметить новые работы
2. Быстрая навигация к конкретным фото
3. Поделиться конкретным фото

---

## Технические Требования

### Браузеры
- Chrome, Firefox, Safari, Edge (последние 2 версии)
- Mobile Safari & Chrome

### Производительность
- Загрузка: < 3 секунд
- Галерея: 60fps
- Изображения: lazy loading, оптимизация

### Хостинг
- Статический хостинг (Netlify/Vercel)
- CDN для изображений

---

## Метрики Успеха

1. **Вовлечённость:** Время на сайте > 2 минут
2. **Исследование:** Просмотр > 5 фото за сессию
3. **Конверсия:** Клики на контакт
4. **Техника:** Нет жалоб на производительность

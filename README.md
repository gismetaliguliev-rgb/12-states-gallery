# KIS-GALLERY - 3D Photography Portfolio

A virtual 3D gallery website for showcasing photography work. Navigate through an immersive space and experience photographs as art pieces on gallery walls.

**Built with ABC-TOM Framework:**
> "Feed the ABC. Let TOM execute."

---

## Project Structure

```
KIS-GALLERY/
├── A-agents/           → Development workflow agents
├── B-brain/            → Research, references, inspiration
├── C-core/             → Project requirements & specs
├── T-tools/            → Components, CMS, workflows
├── O-output/           → Build outputs
├── M-memory/           → Development learnings
├── src/                → Source code
│   ├── js/             → Three.js & gallery logic
│   ├── css/            → Styles
│   └── assets/         → Photos, 3D models
```

---

## Features

### 3D Virtual Gallery
- Immersive 3D space built with Three.js
- First-person navigation (WASD + mouse)
- Photographs displayed as framed art pieces
- Smooth camera transitions
- Mobile-friendly touch controls

### CMS System
- Easy photo upload & management
- Drag-and-drop positioning in gallery
- Photo metadata (title, description, date)
- Gallery room configuration
- No coding required to update content

### Performance
- Lazy loading for images
- Optimized 3D rendering
- Progressive enhancement
- Works on all modern browsers

## Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Start CMS admin
npm run cms
```

---

## Tech Stack

- **Three.js** - 3D rendering
- **Vite** - Build tool
- **JSON** - CMS data storage (upgradeable to database)
- **Vanilla JS** - No framework overhead
